# Daily Habit Tracker

## Overview

This project was created as part of Week 3 of the JavaScript module.

The Daily Habit Tracker allows users to create daily habits, mark them as completed, track their current streak, and view a summary of their progress. The application runs entirely in the browser using HTML, CSS, and JavaScript.

---

## Features

- Add a new habit
- Set a target number of days per week
- Select a habit category
- Form validation with error messages
- Mark habits as completed for the day
- Track habit streaks
- Delete habits
- View a progress summary
- Calculate completion percentage automatically

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## How to Use

1. Enter a habit name.
2. Enter a target number of days between 1 and 7.
3. Select a category.
4. Click **Add Habit**.
5. Tick the checkbox when the habit is completed.
6. Use the **Delete** button to remove a habit.

- id
- name

- target
